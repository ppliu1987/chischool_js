课前预习：
	1.文件
	2.文件扩展名
		文件的组成=文件名（给人看的）+扩展名(也叫后缀)（给电脑看的）
		.后缀是不影响文件的,只影响电脑去读这个文件
		.pptx .xlsx .docx .txt  .jpg .mp3 .mp4  .html .htm  .jsp  自定义扩展名 .aaa .bbb
	3.编辑器
	4.标签
		标签是什么,标签就是一个给浏览器看的标记,通过这些标记,浏览器可以将标签包裹的内容以相应的形式呈现出来
	5.标签分类
		单标签和双标签
	6.具体标签学习
		<!DOCTYPE html>注意1.叹号2.大写 小写
		<!DOCTYPE> 声明必须是 HTML 文档的第一行，位于 <html> 标签之前。
		<!DOCTYPE> 声明不是 HTML 标签；它是指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令。
		<html>--跟标签 只能有一个  所有的只能写它里面
		<head>--头标签 只能有一个  他里面的内容是展示给电脑看的
		<body>--体标签 只能有一个  他里面的内容是展示给人看的
		<meta>  不写meta展示乱码
		<title>--只显示第一个
		<!-- -->注释的作用:1.添加代码说明,提高代码的易读性2.使代码无效
		<br/> blank row
		<hr/>horizontal rule
		<h1>--<h6>独占一行,字体加粗,字体变大
		<p>this is a paragraph!</p>独占一行
	7.单双标签的寓意讲解
